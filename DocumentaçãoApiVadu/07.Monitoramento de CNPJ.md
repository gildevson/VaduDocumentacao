# 7.0 - Monitoramento de CNPJ

Esta API permite realizar operações de:

* Inclusão de CNPJs em grupos de monitoramento;
* Consulta de grupos de monitoramento;
* Consulta de CNPJs monitorados;
* Atualização de grupos de monitoramento;
* Exclusão de CNPJs monitorados.

---

# Passo 01 – Obter Token de Acesso

Antes de utilizar os endpoints de monitoramento, é necessário obter um token temporário de acesso.

## Configuração da Requisição

Na aba **Authorization**:

| Campo | Valor                       |
| ----- | --------------------------- |
| Type  | Bearer Token                |
| Token | API Key fornecida pela VADU |

Após configurar:

1. Clique em **Send**;
2. Aguarde a resposta da API.

## Resultado

Na aba **Body** da resposta será retornado um token temporário.

Exemplo:

```text
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

> Copie o token completo, sem aspas.

### Validade

O token possui validade de:

```text
24 horas
```

Este token será utilizado nos próximos passos.

---

# Passo 02 – Consultar Grupos de Monitoramento

Esta operação retorna todos os grupos de monitoramento cadastrados para sua conta.

## Resultado

A resposta retornará informações semelhantes a:

```json
[
  {
    "id": 6421,
    "descricao": "Grupo Principal"
  }
]
```

Anote o valor do campo:

```text
id
```

Ele será utilizado nas próximas etapas.

---

# Passo 03 – Consultar os CNPJs de um Grupo

Utilize o ID obtido na etapa anterior.

## Exemplo de URL

```text
https://www.vadu.com.br/vadu.dll/ApiMonitoramento/Cadastros/6421
```

Onde:

```text
6421
```

é o ID do grupo.

---

## Configuração da Autenticação

Na aba Authorization:

| Campo | Valor                    |
| ----- | ------------------------ |
| Type  | Inherit auth from parent |

Dessa forma o endpoint utilizará automaticamente o token temporário configurado na Collection.

---

## Resultado

Ao clicar em **Send**, será retornada a lista de CNPJs cadastrados naquele grupo.

Exemplo:

```json
[
  {
    "cnpj": "12345678000199"
  },
  {
    "cnpj": "98765432000155"
  }
]
```

---

# Passo 04 – Incluir CNPJs em um Grupo de Monitoramento

Nesta etapa será utilizado o mesmo ID do grupo na URL.

## Método

```http
POST
```

## URL

```text
https://www.vadu.com.br/vadu.dll/ApiMonitoramento/Cadastros/{ID_GRUPO}
```

---

## Body da Requisição

Informe a lista de CNPJs que deseja adicionar ao grupo.

### Exemplo

```json
[
  "12345678000199",
  "98765432000155"
]
```

---

## Resultado

Após clicar em **Send**, a API retornará:

* CNPJs importados com sucesso;
* CNPJs já existentes;
* CNPJs inválidos.

### Exemplo

```json
{
  "importados": [
    "12345678000199"
  ],
  "existentes": [
    "98765432000155"
  ],
  "invalidos": [
    "11111111111111"
  ]
}
```

---

# Passo 05 – Atualizar os CNPJs de um Grupo

Esta operação substitui completamente a lista atual de CNPJs do grupo.

## Método

```http
PUT
```

## Importante

Os CNPJs existentes que não forem enviados na nova lista serão removidos automaticamente do grupo.

---

## Body da Requisição

Exemplo:

```json
[
  "12345678000199",
  "98765432000155",
  "11222333000144"
]
```

---

## Resultado

A API retornará:

* Lista atualizada;
* CNPJs inválidos;
* Situação da atualização.

### Observação

Os CNPJs inválidos não serão incluídos.

Os CNPJs antigos que não estiverem na nova lista serão removidos automaticamente.

---

# Passo 06 – Excluir CNPJs de um Grupo de Monitoramento

Também é possível remover apenas alguns CNPJs específicos do grupo.

## Método

```http
DELETE
```

## URL

```text
https://www.vadu.com.br/vadu.dll/ApiMonitoramento/Cadastros/{ID_GRUPO}
```

---

## Body da Requisição

Informe a lista dos CNPJs que deseja excluir.

### Exemplo

```json
[
  "12345678000199",
  "98765432000155"
]
```

---

## Resultado

Após clicar em **Send**, a API retornará os CNPJs removidos com sucesso.

### Exemplo

```json
{
  "excluidos": [
    "12345678000199",
    "98765432000155"
  ]
}
```

---

# Resumo dos Métodos

| Operação                 | Método HTTP |
| ------------------------ | ----------- |
| Obter Token              | GET         |
| Consultar Grupos         | GET         |
| Consultar CNPJs do Grupo | GET         |
| Incluir CNPJs            | POST        |
| Atualizar Lista de CNPJs | PUT         |
| Excluir CNPJs            | DELETE      |

---

# Fluxo Completo

1. Obter Token Temporário;
2. Consultar os grupos disponíveis;
3. Selecionar um grupo pelo ID;
4. Consultar os CNPJs existentes;
5. Adicionar novos CNPJs;
6. Atualizar a lista quando necessário;
7. Remover CNPJs quando desejado.
